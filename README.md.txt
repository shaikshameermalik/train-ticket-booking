# Train Ticket Booking System

## Description
A web application to book train tickets with seat reservation functionality. It consists of a front-end built with React and a back-end API using Node.js and PostgreSQL.

## Features
- User authentication
- Seat availability and booking
- View and delete booking history
- Frontend and Backend Integration

## Technologies Used
- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL
- Authentication: JWT (JSON Web Token)

## How to Run

### Backend
1. Clone the repository.
2. Go to the backend folder: `cd backend`.
3. Install dependencies: `npm install`.
4. Run the server: `npm start`.

### Frontend
1. Go to the frontend folder: `cd frontend`.
2. Install dependencies: `npm install`.
3. Run the React app: `npm start`.

### Database
- Set up the PostgreSQL database using the provided schema.
